# 智控视界：UDP透传赋能的机器人即时影像传输与智能遥控平台设计

> 本项目为2024年度国家级大学生创新训练计划项目，项目组成员为：
> - 费腾
> - 王宇恒
> - 郭宇

### 技术路线：
- 采用UDP协议进行机器人数据传输，实现遥控
- 采用webrtc协议进行图像采集，实现视频流传输
